# Quiz app

## Clone this repository
```
git clone git@github.com:llinards/vuejs-quiz-app.git
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
